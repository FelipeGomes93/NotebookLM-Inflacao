# NotebookLM-Inflacao
Este repositório contém um caderno temático desenvolvido no NotebookLM sobre o tema inflação e seus efeitos no poder de compra.
O projeto foi elaborado como parte de um desafio de bootcamp e reúne fontes confiáveis em texto e vídeo, além de materiais de aprendizagem gerados com apoio de IA, incluindo:
mapa mental, podcast, apresentação de slides, vídeo-resumo. O objetivo é demonstrar o uso da inteligência artificial como ferramenta de aprendizagem ativa, aliando curadoria de fontes, pensamento crítico e organização do conhecimento em educação financeira.
